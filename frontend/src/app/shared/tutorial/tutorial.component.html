Lets Get Started!


Start by adding a Machine. It's not going to give you any options for the job field, but that is perfectly normal.

You will need to have at least 1 machine to be able to add a Part. Each Part needs to be assigned a "job number," which will give you the ability to asign a job to a machine later.

Now that you have created a machine, you can add your first part! A Part needs to have a "part number," "job number," and "machine." all other fields are optional. The same Part can be used more than once, where it is ok to have either the same "job number" or "machine." However the same combination of "job number" and "machine" cannot repeat.

Now that you have created your first part, you will be able to use the production calculators, and production tracking features for that part.

Lets start by using the Hourly Tracking where you will see a button that allows you to asign a job to you machine if you have not already done so. Once a job is assigned, you can add a production count by clicking on "quick input."

Now lets try adding a Production Lot. Your production lots will be used to track how many parts have been made to the job so far.

After adding Multiple Production Lots for one job, the Production By Job page will show total parts to date for the job.
